ds Python

Run dev scripts from your project's existing config file. One command for package.json, pyproject.toml, Cargo.toml, and more.

cosmofy Python

Build portable, single-file Python executables using Cosmopolitan libc. Runs on Windows, macOS, and Linux without installation.

attrbox Python

Attribute-based data structures. Access dict keys as attributes with dot notation for cleaner, more readable code.

[`castfit 0.1.1`][release] is available. `castfit` is my tiny library for converting things (mostly `dict` of keys) into classes using standard python type hints.

To install `castfit`:

```bash
# modern (recommended)
uv add castfit

# classic
python -m pip install castfit
```
